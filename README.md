## 프로젝트 소개

어썸리드는 **현장에서 발굴한 문제로 기업을 설득하고, 기술로 실질적 변화를 이끌어내는** 사내 관리 시스템입니다.

2022년 (주)어썸리드 현장직 근무 당시 발견한 문제를 디지털 전환(DX) 과제로 정의한 시스템으로, 기업의 출입문을 통과하는 순간부터 퇴근까지 전 과정의 불편함을 제거하고자 합니다.

### 프로젝트 배경

**"현장에서 발굴한 문제로 기업을 설득하고, 기술로 실질적 변화를 이끌어낸다"**

- 보안이 중시되는 기업 환경에서 수기 명부 작성 시 발생하는 기록 누락과 실명의 어려움을 해결하기 위해 디지털 방문객 관리 체계를 구축
- 수백 명에 달하는 현장 근로자의 안전 교육 이수 현황을 체계적으로 전산화하여 수동 관리의 번거로움을 제거
